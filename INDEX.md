# LUNARIS Document Set — Index & Usage Guide

Drop this folder into the root of a new git repo. Then start Claude Code and say:

> "Read CLAUDE.md, PRD.md and TASKS.md, then begin Milestone 0."

## Reading order (humans)

1. PRD.md — what we're building and why
2. docs/GDD.md — how it plays
3. docs/SDD.md — the physics contract (constants & formulas)
4. docs/TAD.md — the architecture
5. TASKS.md — the build order
